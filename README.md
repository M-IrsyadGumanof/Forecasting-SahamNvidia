📈 Forecasting Saham Nvidia
Proyek ini bertujuan untuk melakukan analisis dan prediksi (forecasting) terhadap harga saham Nvidia menggunakan Metode Regresi Linear.

📊 Dataset
Dataset yang digunakan dalam proyek ini => dataset_Nvidia.xlsx
Berisi data historis saham Nvidia seperti harga pembukaan, penutupan, volume, dan lainnya.

🎯 Tujuan Proyek
1. Menganalisis pola pergerakan harga saham Nvidia
2. Melakukan forecasting harga saham ke depan
3. Mempelajari teknik time series dalam data saham
